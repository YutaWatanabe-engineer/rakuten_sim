実は今と全く同じ暮らしを「楽天経済圏」に
変更して、楽天証券で楽天VT全世界株式インデックファンドを
毎月買っているだけで、人によっては数百万円の
資産になる。

知らず知らずに機械費用が数百万円に膨れあがっている。

数百万円あれば、田舎で暮らせば、数年生きることが
できる。

#プロダクト

そして、現在の年齢を入力してもらう。

積立投資信託をした場合のシミュレーションと、
「100%楽天経済圏」に入った場合のシミュレーションを
グラフにする。
